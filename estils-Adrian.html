ESTILOS HTML

Etiquetas

Es el lenguaje el cual se escribe HTML, en css se pueden usar alguna etiquetas para otorgar estilos a elementos y textos.

ID

La id es un atributo que especifica una unica id para un elemento HTML, el valor tiene que ser unico en el documento. El atributo id se usa mayormente para anotar un estilo en la tabla de estilos y posteriormente ser manipulado por Javascript.

Class

El atributo class hace posible definir de manera igual estilos de elementos <div> iguales.
