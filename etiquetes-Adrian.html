Etiquetas para imagenes

Etiqueta Descripcion
<img src=...> La fuente de donde se muestra la imagen.
<img src=... alt="..."> Se mostrara un texto si la imagen no carga.

Etiqueta para enlaces

Etiqueta Descripcion
<A HREF="URL"></A> Crea un hiperenlace a la direccion puesta.

Etiquetas para ordenar contenido

Etiqueta Descripcion
<DIV> Representa un contenedor generico sin ningun significado especial.
<figure> Representa una figura ilustrada como parte del documento.
<p> Define una parte que debe mostrarse como un parrafo.
<pre> Indica que su contenido esta preformateado y que este formato debe ser preservado.

Etiqueta para citas

Etiqueta Descripcion
<blockquote> Crea un bloque para una cita.

Etiquetas para listas

Etiqueta Descripcion
<ol> Define una lista ordenada de articulos.
<ul> Defina una lista de arituculos sin orden.
<li> Define un articulo de una lista enumerada.
